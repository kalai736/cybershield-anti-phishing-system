# 🛡️ CyberShield – Anti-Phishing System

## About

CyberShield is a simple web application that checks whether a URL is safe or suspicious. It helps users identify phishing websites through a clean and user-friendly interface.

## Features

- URL Security Check
- Suspicious Link Detection
- Instant Result Display
- Responsive User Interface

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages

## How to Use

1. Open the website.
2. Enter a website URL.
3. Click the **Scan** button.
4. View the security result.
