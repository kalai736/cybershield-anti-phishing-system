## About the Project

CyberShield – Anti-Phishing System is a web application designed to detect phishing URLs and suspicious email content. It helps users identify potential cyber threats and provides instant security alerts through a simple and user-friendly interface.

### Key Features
- Detects phishing URLs
- Analyzes suspicious emails
- Instant security alerts
- Responsive and user-friendly design

### Technologies Used
- HTML
- CSS
- JavaScript
- GitHub Pages

### Project Goal
To improve online security by helping users recognize and avoid phishing attacks.
