# Changelog

All notable changes to this project will be documented in this file.

## Version 1.0.0 - Initial Release

### Added

* User registration and login system.
* Website phishing detection module.
* Email phishing analysis module.
* URL verification and validation module.
* Threat detection using rule-based techniques.
* Real-time alert and notification system.
* Threat logging and reporting module.
* MySQL database integration.
* Responsive user interface using HTML, CSS, and Bootstrap.
* Python Flask backend for application logic.
* GitHub project documentation and README.

### Fixed

* Initial bug fixes and performance improvements.

### Security

* Basic phishing detection using predefined security rules.
* URL safety verification.
* Improved handling of suspicious website and email inputs.
